Deploy GitHub to Production:

You are about to launch the deployment of a Flask application to the Production environment. Few steps in the process:
  • Pulling the code from GitHub
  • Building an Image
  • Image deployment to Slave machine for Seleniumn tests
  • If all tests are successful: Loading Image to Docker Hub
  • Auto-Deployment of a running Web-Application to the Production machines
